# 🛒 Kupon Avcısı / Coupon Hunter

Bu proje, Trendyol sepet sayfasında indirim kodlarını otomatik olarak deneyen ve fiyat değişimlerini izleyen bir otomasyon aracıdır.
This project is an automation tool designed to automatically test coupon codes on the Trendyol basket page and monitor price changes.

---

### 🇹🇷 Türkçe Açıklama

**Proje Hakkında:**
Bu araç, **Yapay Zeka (AI)** desteği ile geliştirilmiştir. Manuel olarak girilmesi gereken onlarca kupon kodunu saniyeler içinde sepetinizde dener ve tutar düşüşü gerçekleştiğinde sizi bilgilendirir.

**⚠️ Önemli Uyarılar:**
* **Kupon Güncelleme:** Script içerisinde güncel kupon bulunmaz. Çalışan kuponları kendiniz bulup kodun başındaki listeye eklemelisiniz.
* **Sorumluluk:** Kullanım sırasında oluşabilecek hesap kısıtlamaları (ban) veya teknik sorunlardan geliştirici sorumlu tutulamaz. Risk kullanıcıya aittir.
* **Kullanım:** Sadece eğitim ve test amaçlıdır.

**🚀 Nasıl Çalışır?**
1. Trendyol **Sepetim** sayfasını açın.
2. `F12` ile **Console** sekmesine girin.
3. `coupon-bot.js` kodunu yapıştırıp `Enter`a basın.

---

### 🇺🇸 English Description

**About the Project:**
This tool was developed with the assistance of **Artificial Intelligence (AI)**. It automatically tests dozens of coupon codes in your cart within seconds and notifies you when a price drop is detected.

**⚠️ Critical Disclaimers:**
* **Coupon Updates:** The script does not provide active coupons. You must find valid coupons and update the list in the code manually.
* **Responsibility:** The developer is NOT responsible for any account bans, restrictions, or technical issues. Use at your own risk.
* **Usage:** For educational and testing purposes only.

**🚀 How to Use?**
1. Go to your Trendyol **Cart**.
2. Press `F12` and open the **Console** tab.
3. Paste the code from `coupon-bot.js` and hit `Enter`.
